# FRONT

## How to run

* Clone this repo in your machine

* **With Docker**

  * Run front-end in one terminal:

        $ docker-compose up

* Wait until see this message in the another terminal:

      App running at:
         - Local:   http://localhost:8080/

* In your browser, access this page http://localhost:8080/login
